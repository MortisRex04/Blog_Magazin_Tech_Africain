#📰 Blog Magazin Tech Africain
Bienvenue sur le Blog Magazin Tech Africain, un magazine en ligne dédié à l'actualité technologique africaine.

##🎯 Objectif
Proposer une lecture claire, rapide et moderne de l'actualité tech avec une interface utilisateur soignée.

##🚀 Fonctionnalités
Page d'accueil avec articles mis en avant
Grille des articles récents
Sidebar articles tendances
Page de lecture d'articles complète
Section commentaires
Inscription à la newsletter

##🛠️ Technologies utilisées
HTML5 : Structure sémantique multipage (Accueil, Dashboard, Carte, Prévisions, Comparaison, À propos).
CSS3 : Styles modernes (Flexbox/Grid), variables CSS, typographie importée (Google Fonts : DM Sans, Space Grotesk) et animations sur mesure.
JavaScript (Vanilla) : Gestion de la logique d'animation (Intersection Observer), interactions du DOM et simulation des données.

##🚀 Installation et exécution
Ce projet est purement front-end (HTML/CSS/JS). Aucun serveur backend complexe n'est requis pour le tester localement.

Cloner le dépôt
git clone https://github.com/MortisRex04/Blog_Magazin_Tech_Africain
Ouvrir le projet Allez dans le dossier du projet :
cd Blog_Magazin_Tech_Africain
Lancer l'application Ouvrez le fichier index.html directement dans votre navigateur web préféré (Chrome, Firefox, Safari, Edge), ou utilisez une extension comme Live Server sur VS Code pour une meilleure expérience d'itération.
